# 🛒 Online Grocery Delivery System (SQL Database)

A relational **Online Grocery Delivery System** database developed as a **Database Systems Lab** project. The project is designed using SQL and demonstrates database design, normalization, relational modeling, and query implementation for managing an online grocery platform.

---

## 📌 Project Overview

The Online Grocery Delivery System is designed to efficiently manage customers, products, suppliers, orders, payments, deliveries, shopping carts, and reviews. The project focuses on designing a well-structured relational database with proper constraints, relationships, and SQL operations.
